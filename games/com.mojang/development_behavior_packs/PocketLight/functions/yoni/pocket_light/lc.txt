玩家
  当 物品 切换
    更新 plight

  当 plight 更新
    如果 没有 光点
      创建 光点
    向上 同步 plight

光点
  设置 flag0 = 0
  
  当 plight 更新
    更新光块
  
  当 更新光块
    如果 flag0 = 0 且 位置 有 光块
      设置 flag0 = -1
    否则
      设置 flag0 = 1
    如果 flag0 = 1
      如果 位置 有 光块
        移除光块
      放置 光块 亮度 plight
    
  当 没有 玩家
    设置 plight = 0
  
  当 光度 0
    移除自己
    
  当 移除自己
    如果 flag0 = 1
      清理 光块
    kill @s
